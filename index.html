Summary of HTML 5

HTML 5 stands for Hyper Text Markup Language, the 5 means it is the 5th version of HTML. One of HTML 5's main goals is to improve browser, computer, and human readability. But another goal of HTML 5 is to improve support for the latest multimedia. HTML 5 has many goals and is still in development it will continue to be developed for many years to come. It has the ability to use apps even when the browser isn't connected to the internet. This is because HTML 5 also utilizes better local storage so apps can be stored locally on your computer. Better connections for gaming with socket based networking. New tags for greater semantics so browser, developer, computer, and user know whats happening. Even with all these new features HTML 5 does not sacrifice quality for quantity, HTML 5 is faster than ever. HTML 5 will become even more stable and useful over time.

Features of HTML 5

  • Offline web apps
  • Timed media playback
  • Drag and Drop
  • Geo Location
  • Web SQL database

This is just to name a few of the features and improvements that HTML 5 has to offer.
